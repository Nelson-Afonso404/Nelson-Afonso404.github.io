Como Ver Materiais em Estoque no SAP

Objetivo

Este passo a passo demonstra como visualizar a quantidade de materiais em estoque no SAP.

Pré-requisitos

  • Acesso ao sistema SAP.
  • Conhecimento da transação MMBE.

Passo a Passo

  1. Acessar a transação MMBE

    Na barra de transações, digite MMBE e pressione Enter.

  2. Inserir o código do material

    No campo Material, digite o código do material que você deseja consultar. Se você não souber o código do material, pode usar a pesquisa para encontrá-lo ou Clique Aqui.

  3. Selecionar o centro de estoque

    No campo Centro, digite o código do centro de estoque que você deseja consultar. Se você não souber o código do centro de estoque, pode usar a pesquisa para encontrá-lo.

  4. Visualizar as informações do material

    Pressione Enter para visualizar as informações do material. A tela exibirá diversas informações sobre o material, incluindo a quantidade disponível em estoque. A quantidade disponível em estoque é exibida no campo Estoque livre.

  5. Analisar as informações do material

    Você pode analisar outras informações do material, como:

    • Estoque em Transferência: Quantidade de material que está em processo de transferência para outro centro de estoque.
    • Estoque Reservado: Quantidade de material que está reservada para um pedido de venda ou produção.
    • Estoque Bloqueado: Quantidade de material que está bloqueada para uso.

  6. Exportar os dados para um arquivo (opcional)

    Se você desejar exportar os dados para um arquivo, clique no botão Exportar. Você pode escolher exportar os dados para um arquivo Excel, CSV ou outro formato.

Dicas

  • Você pode usar a transação MB52 para visualizar a disponibilidade de um material em diferentes centros de estoque e períodos.
  • Você pode usar a transação MM03 para visualizar o cadastro completo de um material.
